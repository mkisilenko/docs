## Popular Combinations

<AccordionGroup>
<Accordion title="SaaS Startup MVP">
  **What to build:**
  1. Landing page (waitlist + demo video)
  2. SaaS app (core features + auth)
  3. Simple dashboard (user metrics)
  
  **Time:** 40-50 minutes total
  
  **Database:** Anyx managed (easiest)
  
  **AI Features:** Lead assistant on landing page, insights in dashboard
</Accordion>

<Accordion title="Creator Portfolio + Blog">
  **What to build:**
  1. Landing page (portfolio showcase)
  2. Content site (blog posts)
  3. Contact form
  
  **Time:** 25-30 minutes total
  
  **Database:** None needed (or minimal for contact form)
  
  **AI Features:** Reading assistant for blog content
</Accordion>

<Accordion title="E-commerce Store">
  **What to build:**
  1. Product catalog (main store)
  2. Landing page (homepage marketing)
  3. Simple dashboard (order tracking)
  
  **Time:** 35-40 minutes total
  
  **Database:** Anyx managed or your Supabase
  
  **AI Features:** Shopping assistant, visual search
</Accordion>

<Accordion title="Educational Platform">
  **What to build:**
  1. Content site (course materials)
  2. Quiz game (practice exercises)
  3. Simple dashboard (progress tracking)
  
  **Time:** 35-45 minutes total
  
  **Database:** Anyx managed (for user progress)
  
  **AI Features:** Reading assistant, AI tutor in game
</Accordion>
</AccordionGroup>
