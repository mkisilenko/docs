## Example Prompt: Task Manager

```
"Create a task management tool for freelancers juggling multiple clients.

Target: Solo freelancers and small agencies
Style: Calm, minimal, Notion-inspired. Dark/light mode support

Core features:
- Kanban board (To Do, In Progress, Done columns)
- Task cards with: title, client name, due date, priority, description
- Quick add via command palette (Cmd+K)
- List view and calendar view
- Filter by client, priority, or due date
- Search across all tasks

Authentication:
- Sign up with email + password
- Social login (Google, GitHub)
- Password reset flow
- Email verification

User management:
- User profile page
- Settings (notifications, timezone)
- Subscription plans (Free, Pro $15/mo)

Database: Anyx managed

Add AI command palette for natural language task creation:
- 'Schedule meeting with Sarah next Tuesday 2pm' → creates task
- 'Show overdue tasks' → filters view
- 'Remind me about X tomorrow' → sets reminder"
```

**Build time:** 15-20 minutes
